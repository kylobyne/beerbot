# 🍺 BEER Bot — Развлекательный Telegram-бот

[![Python Version](https://shields.io)](https://python.org)
[![Aiogram](https://shields.io)](https://aiogram.dev)
[![Database](https://shields.io)](https://sqlite.org)

Интерактивный игровой бот для групповых чатов Telegram, написанный на **Aiogram**. Позволяет участникам соревноваться в количестве «выпитого» виртуального пива, вести статистику и бороться за звание главного пивного ценителя чата.

---

## ✨ Возможности

*   **⚡ Команда `/beer`** — Пользователь "выпивает" рандомное количество пива (от **0.1 до 5.0 литров**).
*   **⏱️ Cooldown** — использовать команду можно не чаще одного раза в час.
*   **📊 Команда `/stats`** — отображение таблицы лидеров текущего чата (автоматическое разбиение списка участников на страницы по 20 строк.)
*   **🗄️ Изолированные базы данных** — для каждого чата автоматически создается своя независимая БД в папке `sqlite3/`.

---

## 🛠️ Требования к окружению

*   **Python** версии `3.10` или новее.
*   Токен Telegram-бота, полученный у [@BotFather](https://t.me/BotFather).

---

## 🚀 Быстрый старт

### 1. Клонирование и подготовка
Откройте терминал/PowerShell в папке с проектом и разверните виртуальное окружение:

```powershell
# Создание виртуального окружения
py -m venv .venv

# Активация окружения (Windows PowerShell)
.\.venv\Scripts\Activate.ps1
```

> 💡 **Если PowerShell блокирует запуск скрипта**, выполните команду ниже (только для текущего окна), а затем повторите активацию:
> ```powershell
> Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
> ```

### 2. Установка зависимостей
Установите все необходимые библиотеки одной командой:
```powershell
pip install -r requirements.txt
```

---

## ⚙️ Настройка конфигурации

1. Найдите в корне проекта файл `.env`.
2. Укажите в нём ваш секретный токен:

```env
BOT_TOKEN=1234567890:ABCdefGhIJKlmNoPQRsTUVwXyZ
```

⚠️ **Важно:** Никогда не публикуйте файл `.env` в открытый доступ и не отправляйте его в Git-репозиторий. В нем хранятся конфиденциальные данные вашего бота.

---

## 🏃‍♂️ Запуск бота

Убедитесь, что виртуальное окружение активно, и запустите главный файл:

```powershell
py bot.py
```

*Для остановки работы бота нажмите сочетание клавиш `Ctrl + C` в окне терминала.*
